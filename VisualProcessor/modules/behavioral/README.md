# 👤 Behavior Analysis Module

## 📌 Описание

Модуль для комплексного анализа поведения людей в видео, включая детекцию, трекинг, оценку позы и анализ активности.

## ✅ Реализованные возможности

### Базовые возможности
- ✔ **1. Детекция и трекинг человека** — YOLO11 с BoT-SORT/ByteTrack
- ✔ **2. Pose Estimation** — YOLO11-Pose, MediaPipe Pose, HRNet
- ✔ **3. Hand Pose** — MediaPipe Hands, FrankMocap
- ✔ **4. Head Pose Estimation** — MediaPipe Face Mesh
- ✔ **5. Eye tracking** — специализированные айтрекеры

### Продвинутый анализ поведения (новое)
- ✔ **6. Детальная классификация жестов рук** — 12+ типов жестов (pointing, open_palm, hands_on_hips, self_touch, fist, thumbs_up/down, victory, ok, rock, call_me, love)
- ✔ **7. Body Language анализ** — открытая/закрытая поза, power pose, напряженность, расслабленность, наклон тела, баланс
- ✔ **8. Speech-driven behavior** — анализ синхронности губ, активность речи, параметры рта
- ✔ **9. Engagement Index** — индекс вовлеченности на основе зрительного контакта, движений головы, активности жестов, позы
- ✔ **10. Confidence/Dominance Index** — индекс уверенности и доминантности на основе позы, жестов, положения головы и плеч
- ✔ **11. Stress/Anxiety Detection** — детекция признаков стресса (частое моргание, self-touch, закрытая поза, напряженность, fidgeting, несинхронные движения)

## 🎯 Рекомендуемые модели

| Задача | Рекомендуемые модели / подходы |
|--------|-------------------------------|
| 1. Детекция и трекинг человека | YOLO11 с BoT-SORT/ByteTrack |
| 2. Pose Estimation (поза тела) | YOLO11-Pose (общий баланс), MediaPipe Pose (мобильные/CPU), HRNet (максимальная точность) |
| 3. Hand Pose (поза руки) | MediaPipe Hands (2D/3D), FrankMocap, Awesome Hand Pose |
| 4. Head Pose Estimation | MediaPipe Face Mesh, проекты на базе dlib/OpenCV |
| 5. Eye tracking | Специализированные айтрекеры (Tobii, Pupil Labs), алгоритмы на основе OpenCV/Dlib |

## 🔍 1. Анализ глаз и взгляда (Eye Tracking & Gaze Estimation)

### MediaPipe Iris / Gaze

**Что:** Часть Face Mesh, но можно использовать отдельно

**Задачи:**
- Определение направления взгляда (куда смотрит человек)
- Открытость/закрытость глаз (моргание, сонливость)
- Расстояние до объекта (по размеру радужной оболочки)

### OpenCV + Dlib с 68 facial landmarks

**Преимущество:** Легче для кастомизации

**Метрики:**
- **EAR (Eye Aspect Ratio)** — для определения моргания
- **PERCLOS** — процент времени с закрытыми глазами (для детекции усталости)
- **Пупиллометрия** — отслеживание зрачков

### Специализированные библиотеки

- **PyGaze, ITU GazeTracker** — для точного трекинга взгляда
- **Tobii, Pupil Labs** — с аппаратными решениями

## 🚶 2. Анализ перемещения и трекинг (Movement Analysis & Tracking)

### Мультиобъектный трекинг (MOT)

- **DeepSORT** — золотой стандарт, сочетает YOLO + трекинг
- **ByteTrack** — очень быстрый, используется в Ultralytics YOLO
- **OC-SORT** — устойчив к окклюзиям (когда объекты перекрываются)

### Re-ID (Re-Identification)

**Что:** Идентификация одного человека после перекрытия или выхода из кадра

**Библиотеки:** FastReID, Torchreid

**Применение:** Подсчет уникальных посетителей в магазине

### Траектория и анализ поведения

- **OpenCV + Kalman Filter** — предсказание движения
- **Trajectory clustering** — группировка похожих путей движения

## 🧠 3. Анализ активности и поведения (Activity Recognition)

### Трехмерные сверточные сети (3D CNN)

- **I3D (Inflated 3D ConvNet)** — для распознавания действий
- **SlowFast Networks** — два потока (медленный для контекста, быстрый для движения)

### Двухпотоковые архитектуры

**Что:** Обрабатывает отдельно RGB (внешность) и Optical Flow (движение)

**Библиотеки:** MMAction2, VideoMAE

### Ключевые действия для проекта

- Стояние/сидение/лежание
- Поднятие рук (просит помощи)
- Падение (fall detection)
- Агрессивные жесты

## 📊 4. Продвинутая аналитика сцены

### Depth Estimation (Оценка глубины)

- **MiDaS, DPT** — из одного RGB кадра
- **Stereo vision** — из двух камер

**Применение:** Определение расстояния до объектов

### Optical Flow (Оптический поток)

- **Farneback, Lucas-Kanade** (классические)
- **RAFT, FlowNet** (нейросетевые)

**Применение:** Анализ направлений движения толпы

### Crowd Analysis (Анализ толпы)

- **Плотность толпы** — Counting objects in dense crowds
- **Аномальное поведение** — Выявление драк, паники
- **Social Force Models** — Моделирование поведения толпы

## 🔍 5. Специализированные детекторы

### Fall Detection (Детекция падений)

Сочетание: поза (близко к земле) + резкое движение + отсутствие движения после

### Fight Detection (Детекция драк)

Быстрые резкие движения нескольких людей вблизи

### Аномальные позы

Лежание на полу в публичном месте

### Loitering Detection (Детекция шатания)

- Человек долго находится в одном месте без цели
- Анализ траектории (круговые движения)

## 🛠️ 6. Готовые фреймворки и платформы

### NVIDIA DeepStream

**Плюсы:** Оптимизация под GPU, готовые пайплайны

**Что включает:** Детекция, трекинг, классификация действий

### Microsoft Azure Video Analyzer

**Плюсы:** Облачное решение, минимум кода

**Возможности:** Распознавание объектов, действий, эмоций

### Google MediaPipe (расширенный)

- **Holistic** — объединяет позу, лицо и руки
- **Objectron** — 3D детекция объектов
- **KNIFT** — трекинг и распознавание объектов

## 📈 7. Аналитические метрики

### Для анализа глаз

- **Fixation duration** — сколько времени взгляд фиксируется на объекте
- **Saccade velocity** — скорость движения глаз между точками
- **Pupil dilation** — расширение зрачков (интерес/стресс)

### Для анализа движения

- **Dwell time** — время нахождения в зоне
- **Path efficiency** — оптимальность маршрута
- **Interaction frequency** — частота взаимодействия с объектами

### Для анализа поведения

- **Activity heatmaps** — тепловые карты активности
- **Sequence mining** — поиск паттернов поведения
- **Anomaly score** — оценка аномальности поведения

## 🚀 Использование

### CLI интерфейс

```bash
python main.py --input video.mp4 --output results.json --frame-skip 1
```

**Параметры:**
- `--input, -i` — путь к входному видео файлу (обязательно)
- `--output, -o` — путь к выходному JSON файлу (по умолчанию: `input_name_behavior.json`)
- `--frame-skip` — обрабатывать каждый N-й кадр (по умолчанию: 1)
- `--visualize` — сохранять визуализацию результатов
- `--visualize-dir` — директория для визуализаций (по умолчанию: `./behavior_visualizations`)

### Программный интерфейс

```python
from behavior_analyzer import BehaviorAnalyzer

# Инициализация
analyzer = BehaviorAnalyzer()

# Обработка видео
results = analyzer.process_video(
    video_path="video.mp4",
    output_path="results.json",
    frame_skip=1
)

# Обработка отдельного кадра
import cv2
frame = cv2.imread("frame.jpg")
result = analyzer.process_frame(frame)
```

## 📊 Детальное описание фичей

### Жесты рук (Hand Gestures)

Модуль распознает следующие типы жестов:
- **pointing** — указание рукой
- **open_palm** — раскрытые ладони
- **hands_on_hips** — руки в боки
- **self_touch** — self-touch жесты (поглаживание, почёсывание — индикатор стресса)
- **fist** — кулак
- **thumbs_up** — большой палец вверх
- **thumbs_down** — большой палец вниз
- **victory** — победа (V)
- **ok** — OK знак
- **rock** — рок (рога)
- **call_me** — позвони мне
- **love** — любовь (сердце)

### Body Language

Анализирует:
- **Поза тела** — standing/sitting
- **Открытая/закрытая поза** — на основе расстояния между запястьями
- **Power pose** — доминантная поза (руки на бедрах, широко расставлены)
- **Напряженность (rigidity)** — прямые плечи, напряженная поза
- **Расслабленность** — противоположность напряженности
- **Наклон вперед/назад** — анализ позиции носа относительно плеч
- **Баланс тела** — смещение центра масс

### Speech-driven Behavior

Анализирует:
- **Ширина/высота рта** — параметры открытия рта
- **Площадь рта** — общая площадь открытия
- **Активность речи** — вероятность того, что человек говорит
- **Соотношение открытия** — отношение высоты к ширине

### Engagement Index

Вычисляется на основе:
- **Зрительный контакт** — направление взгляда к камере
- **Движения головы** — активность микродвижений
- **Активность жестов** — количество и частота жестов
- **Открытая поза** — способствует вовлеченности

Метрики:
- `engagement_score` — общий индекс (0-1)
- `engagement_variation` — вариативность во времени
- `engagement_peaks` — количество пиков вовлеченности
- `engagement_consistency` — консистентность (1 - вариативность)

### Confidence/Dominance Index

Вычисляется на основе:
- **Открытая поза** — уверенная поза
- **Power pose** — доминантная поза
- **Положение головы** — прямая голова = уверенность
- **Уверенные жесты** — open_palm, pointing, thumbs_up
- **Положение плеч** — прямые плечи = уверенность

Метрики:
- `confidence_score` — индекс уверенности (0-1)
- `dominance_score` — индекс доминантности (0-1)
- `confidence_variability` — вариативность уверенности
- `confidence_peak_moments` — моменты пиковой уверенности

### Stress/Anxiety Detection

Детектирует признаки стресса:
- **Частое моргание** — высокая частота моргания (>30%)
- **Self-touch жесты** — поглаживание, почёсывание
- **Закрытая поза** — скрещенные руки, закрытая позиция
- **Напряженность** — rigid posture, напряженная поза
- **Fidgeting** — ёрзание, быстрые мелкие движения
- **Несинхронные движения** — асинхронность движений рук

Метрики:
- `stress_level` — общий уровень стресса (0-1)
- `anxiety_score` — индекс тревожности (0-1)
- `stress_count` — количество активных индикаторов стресса
- `stress_indicators` — детализация по каждому индикатору
