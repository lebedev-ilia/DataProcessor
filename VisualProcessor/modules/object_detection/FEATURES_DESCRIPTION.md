# Описание фичей модуля object_detection

## 1. YOLO Вариант (если категории объектов неизвестны)

### class
Название класса обнаруженного объекта из предобученной модели YOLO11x (80 классов COCO). Определяется через индекс класса в модели.

### conf
Уровень уверенности модели в детекции объекта (confidence score), значение от 0.0 до 1.0. Объекты с confidence ниже порога (по умолчанию 0.6) отфильтровываются.

### box
Координаты bounding box объекта в формате [x_min, y_min, x_max, y_max] в пикселях исходного изображения. Используется абсолютная система координат.

## 2. OWL-ViT / OWLv2 Вариант (open-vocabulary detection)

### frames - детекции по кадрам

Каждая детекция содержит следующие поля:

#### bbox
Координаты ограничивающего прямоугольника объекта [x_min, y_min, x_max, y_max] после нормализации и обрезки по границам изображения. Координаты проверяются на валидность (min < max).

#### score
Уровень уверенности модели в детекции объекта, полученный через post-processing выходов модели OWL-ViT/OWLv2. Объекты с score ниже порога (по умолчанию 0.6) отфильтровываются.

#### label
Текстовое название категории объекта, соответствующее одному из запросов (text queries), переданных в модель. Модель OWL-ViT/OWLv2 позволяет детектировать объекты по произвольным текстовым описаниям без переобучения.

#### color (опционально)
Доминирующий цвет объекта в формате BGR {"B": int, "G": int, "R": int}, вычисляемый как среднее значение цветов пикселей в области bounding box. Используется для визуальной характеристики объекта.

#### semantic_tags (опционально)
Список семантических тегов объекта (например, "luxury", "danger", "cute", "sport", "food", "technology"), определяемых на основе ключевых слов в label и эвристических правил. Позволяет дополнительно классифицировать объекты по семантическим категориям.

### summary - статистика по всем кадрам

#### total_detections
Общее количество детекций объектов по всем обработанным кадрам. Отражает общую "насыщенность" видео объектами.

#### unique_categories
Количество уникальных категорий объектов, обнаруженных в видео. Показывает разнообразие объектов.

#### category_counts
Словарь с подсчетом количества детекций для каждой категории объектов. Ключ - название категории (label), значение - количество детекций. Позволяет определить наиболее часто встречающиеся объекты.

#### semantic_tag_counts
Словарь с подсчетом количества детекций для каждого семантического тега. Ключ - название тега, значение - количество детекций с этим тегом. Отражает семантическую структуру объектов в видео.

#### brand_detections
Список детекций брендов (логотипов), где каждый элемент содержит frame (номер кадра), brand (название бренда), score (уверенность) и bbox (координаты). Используется для анализа присутствия брендов в видео.

### frame_count
Общее количество обработанных кадров. Используется для нормализации метрик и статистики.
