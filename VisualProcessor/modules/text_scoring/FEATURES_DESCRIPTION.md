# Описание фичей модуля text_scoring

Модуль извлекает фичи текста в кадре через OCR и анализирует их взаимодействие с видео. Использует EasyOCR для детекции текста и вычисляет метрики плотности, оформления, семантики и временных характеристик текста.

## 1. Text → Action / Motion Correlation

### text_action_sync_score
Средняя корреляция между появлением текста и пиками движения в видео. Вычисляется как среднее значение интенсивности движения в моменты появления текста. Показывает, насколько синхронно текст появляется с действиями в кадре.

### text_motion_alignment
Средняя оценка выравнивания текста с мультимодальными пиками (движение + лицо + аудио). Вычисляется как взвешенная комбинация пиков движения (40%), лица (30%) и аудио (30%) в моменты появления текста.

### multimodal_attention_boost_score
Максимальная оценка мультимодального выравнивания текста. Показывает пиковый момент, когда текст максимально синхронизирован с визуальными и аудио сигналами.

## 2. Text Duration and Continuity

### text_on_screen_continuity
Средняя длительность отображения текста на экране в секундах. Вычисляется как среднее значение длительностей всех уникальных текстовых элементов.

### text_switch_rate
Частота смены текста на экране (количество уникальных текстов в секунду). Показывает, как быстро меняется текст в видео.

## 3. Call-to-Action (CTA) Detection

### cta_presence
Бинарный флаг наличия CTA в видео (1 - есть, 0 - нет). Определяется через поиск ключевых слов: "subscribe", "follow", "like", "link in bio", "click", "watch".

### cta_timestamp
Средний временной момент появления CTA в секундах. Вычисляется как среднее значение временных меток всех обнаруженных CTA.

### cta_strength
Средняя сила CTA, вычисляемая как среднее значение мультимодальных оценок в моменты появления CTA. Показывает, насколько эмоционально и визуально выделены моменты призыва к действию.

## 4. Text Emphasis Peaks

### text_emphasis_peak_flags
Список индексов кадров, где текст имеет максимальное выделение (верхние 15% по мультимодальной оценке). Используется для идентификации ключевых моментов с текстом.

## 5. Raw OCR Data

Модуль также сохраняет сырые данные OCR для каждого кадра:
- **frame**: индекс кадра
- **bbox**: координаты текста (x1, y1, x2, y2)
- **text**: распознанный текст
- **confidence**: уверенность OCR (0.0-1.0)
- **is_cta**: флаг, является ли текст CTA

## Методы вычисления

1. **OCR Extraction**: Используется EasyOCR с поддержкой английского и русского языков для детекции текста в кадрах.
2. **Motion/Face/Audio Integration**: Модуль может использовать данные из других модулей (optical_flow, emotion_face, audio processor) для вычисления мультимодальных метрик.
3. **Normalization**: Все сигналы нормализуются и сглаживаются через Gaussian filter для уменьшения шумов.

## Зависимости

- **easyocr**: Библиотека для OCR
- **numpy**: Для численных вычислений
- **scipy**: Для фильтрации сигналов

## Использование

Модуль может работать автономно или интегрироваться с данными из других модулей через флаги:
- `--use-motion-data`: Использовать данные движения из optical_flow
- `--use-face-data`: Использовать данные эмоций из emotion_face
- `--use-audio-data`: Использовать аудио данные
