## Site design and UX (MVP) — полуфинал

Этот документ фиксирует решения по дизайну, структуре и UX сайта TrendFlow.

### 1) Общая концепция дизайна

**Стиль**: тёмная тема, минимализм с техно-акцентами, динамические элементы.

**Цветовая палитра**:
- Фон: `#0a0a0a` (почти чёрный) или `#1a1a1a` (тёмно-серый)
- Карточки/панели: `#1f1f1f` или `#2a2a2a`
- Текст основной: `#e0e0e0` или `#f5f5f5`
- Текст вторичный: `#a0a0a0`
- Акценты фиолетовые: `#8b5cf6`, `#a78bfa`
- Акценты голубые: `#3b82f6`, `#60a5fa`
- Градиенты: от фиолетового к голубому для кнопок и акцентов

**Типографика**:
- Основной шрифт: Inter (Google Fonts)
- Моноширинный (для кода/логов): JetBrains Mono или Fira Code
- Размеры: scale (12px, 14px, 16px, 20px, 24px, 32px, 48px)

**Иконки**: минималистичные, динамические (маленькие анимации), библиотеки Lucide React или Heroicons.

### 2) Главная страница (landing)

**Структура**:
- **Header** (sticky): логотип (динамический) + название слева, меню по центру, кнопки "Анализ"/"ЛК" справа
- **Background**: динамический 3D (Three.js/React Three Fiber на desktop, отключать на мобильных)
- **Hero секция**: большой заголовок, описание, кнопки "Демо"/"Подробнее"
- **Description блоки**: интерактивные секции по модальностям (Visual, Audio, Text) с переключаемыми анимациями
- **Тарифы**: уникальный дизайн карточек (не стандартные прямоугольники)

**3D фон**:
- Технология: Three.js или React Three Fiber
- Варианты: абстрактная сеть/граф, плавающие частицы, геометрические фигуры
- Мобильная версия: статичный градиент или простая CSS анимация
- Оптимизация: `requestAnimationFrame`, LOD, минимум полигонов

**Логотип (динамический)**: CSS animations или SVG animations, плавная анимация каждые 10 секунд.

### 3) Навигация

**Header меню**:
- Для неавторизованных: "Главная", "Документация", "Контакты"
- Для авторизованных: "Анализ", "Личный кабинет" (или иконка)

**Футер**:
- Левая колонка: логотип + описание
- Центральные колонки: "Продукт", "Компания", "Правовая информация"
- Правая колонка: контакты + социальные сети
- Нижняя строка: копирайт

### 4) Создание анализа

**Форма (многошаговая)**:
1. **Источник видео**: табы "YouTube URL" / "Загрузить файл"
2. **Базовая информация**: метаданные (автоматически для YouTube, форма для upload)
3. **Конфигурация**: выбор из готовых или создание новой

**Страница конфигураций**:
- Две секции: "Готовые конфигурации" (наши) + "Мои конфигурации" (пользовательские)
- Карточки с названием, описанием, стоимостью, временем, целевой аудиторией

**Создание конфигурации**:
- **Структура**: левая панель (навигация по модальностям), центральная область (фичи), правая панель (оценка стоимости/времени)
- **Секции**: группировка фичей по модальностям (Visual, Audio, Text, Temporal)
- **Карточка фичи**: чекбокс, название, описание, ссылка на документацию, стоимость/время
- **Оценка**: обновляется в реальном времени при изменении выбора

### 5) Страница результатов анализа

**Структура (Google Console-стиль)**:
- **Верхняя панель**: breadcrumbs, название анализа, действия (экспорт, удалить)
- **Левая панель (sidebar)**: навигация по разделам (Прогноз, Visual, Audio, Text, Рекомендации)
- **Центральная область**: контент выбранного раздела
- **Правая панель (опционально)**: детали выбранного элемента

**Прогноз**:
- По умолчанию: крупные карточки с прогнозом для каждого горизонта (7d, 14d, 21d)
- Переключение вида: кнопки "Карточки" / "Таблица" / "График"

**Графики**:
- Библиотеки: Chart.js, Recharts, D3.js, Plotly.js
- Типы: timeline, bar charts, pie charts, heatmaps
- Расположение: grid layout, карточки, возможность drag & drop

**LLM-разбор**: раздел "Рекомендации" в sidebar, текст в карточке с тёмным фоном.

**Real-time визуализация обработки**:
- WebSocket соединение с backend для получения событий
- Canvas overlay на превью видео с bounding boxes в реальном времени
- Этапы: показывать текущий этап + визуализацию на видео
- Упрощённая версия для MVP: превью кадров с нарисованными bounding boxes после завершения этапа

### 6) Личный кабинет

**Структура (Google Console-стиль)**:
- **Верхняя панель**: баланс (крупно) + кнопка "Пополнить"
- **Левая панель (sidebar)**: навигация (Главная, Анализы, Конфигурации, Биллинг, Настройки)
- **Центральная область**: контент выбранного раздела

**Dashboard (главная)**:
- Карточки метрик: "Всего анализов", "Потрачено", "Активных"
- Графики: "Расходы по времени", "Анализы по статусу"
- Последние анализы: компактный список (5 последних)

**Анализы**:
- Вкладка "Текущие": карточки с превью, прогресс-бар, этап обработки
- Вкладка "Завершённые": карточки с превью, метаданные (дата, статус, стоимость, время, конфигурация)

**Пополнение баланса**:
- Быстрые кнопки: "500₽", "1000₽", "2000₽", "5000₽"
- Поле "Другая сумма"
- Выбор способа оплаты
- Уникальный дизайн карточек (градиенты, hover-эффекты)

### 7) Состояния и интерактивность

**Loading состояния**:
- В списке "Текущие": карточка с превью, прогресс-бар с процентами, текст этапа
- На странице анализа: детальный прогресс-бар, консоль логов, real-time визуализация
- WebSocket для real-time обновлений, polling fallback

**Ошибки и валидация**:
- Toast-уведомления: правый верхний угол, цветовое кодирование (успех=зелёный, ошибка=красный, предупреждение=жёлтый)
- Inline ошибки: под полями ввода (красный текст)
- В логах: цветовое кодирование (info=синий, warning=жёлтый, error=красный)

**Пустые состояния**:
- Центрированные блоки с иконками, текстом, кнопками действий
- Примеры: "Нет завершённых анализов", "Результаты будут доступны после завершения"

### 8) Адаптивность

**Breakpoints**:
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

**Мобильные оптимизации**:
- Упрощённые графики
- Горизонтальный скролл для таблиц
- Hamburger menu вместо полного меню
- Отключение 3D фона
- Вертикальная навигация

### 9) Дополнительные страницы

**"Как это работает"**:
- Пошаговое объяснение с визуализациями
- FAQ секция

**Документация**:
- Система документации (Docusaurus, GitBook, или Markdown с навигацией)
- Структура: "Начало работы", "Руководства", "API", "Фичи", "FAQ"
- Поиск по документации

**Поддержка**:
- FAQ страница
- LLM-бот: чат-виджет в правом нижнем углу, обучен на документации
- Тикет-система: форма обратной связи (fallback)

### 10) Технологический стек (рекомендации)

**Frontend**:
- Next.js (React) + TypeScript
- Tailwind CSS (стилизация)
- shadcn/ui (UI компоненты)

**3D и анимации**:
- React Three Fiber (3D)
- Framer Motion (анимации)

**Графики**:
- Chart.js / Recharts (стандартные графики)
- D3.js (кастомные визуализации)
- Plotly.js (интерактивные графики)

**Дополнительно**:
- WebSocket (real-time обновления)
- react-hot-toast (уведомления)
- Lucide React / Heroicons (иконки)
