# Model licenses inventory (template)

Цель: инвентаризация используемых моделей и их лицензий для коммерческого продукта.

Заполнять по мере добавления моделей/версий.

## Таблица

| model_name | model_version / revision | source (URL) | license | commercial use OK? | notes/constraints |
|---|---|---|---|---|---|
|  |  |  |  |  |  |

## Правило

- Нельзя добавлять модель в прод без строки в этой таблице.
- Для HuggingFace моделей фиксировать `revision` (commit/tag), не “latest”.
